使用案例

  1. git log 显示本地我已經提交了a, b, c三个版本。如果我想回退到b版本，可以直接使用git reset b命令，这样工作空间就被回退到那个版本，通过git status命令可以查看c版本的那些改动全部都变成未提交的了，至于之后要丢弃掉这些更改，还是要做其他的处理根据需要而定。
